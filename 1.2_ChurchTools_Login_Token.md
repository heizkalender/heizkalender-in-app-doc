# ChurchTools Token

Der ChurchTools Login Token dient der sicheren Authentifizierung und Autorisierung der CCU gegenüber ChurchTools.

Der Token wird in der CCU Systemvariable gespeichert: **HK1-CT-Token**

## Hinweis

Den ChurchTools Token erstellen Sie wie folgt:

1) Erstellen Sie in ChurchTools eine neutrale Person (z.B. namens „API-User“) und vergeben Sie die Berechtigungen *verwalten*, *Persons & Groups sehen (view)*, *Calendar* und *Resources freigeben*. 

2) Dann melden Sie sich als diese Person bei ChurchTools an. Wie zuvor wieder auf *Persons & Groups* gehen, die (eigene) API-Person anklicken, Berechtigungen wählen, dann *Login-TokenÜ anklicken.

3) (a) Login-Token herauskopieren und in Ihrem Verwaltungssystem abspeichern, damit Sie immer bequem zugreifen können und (b) den Token in diesem Feld eintragen


**Wichtig: Stelle Sie sicher, dass im ChurchTools die Zeitzoneneinstellung auf „Europe/Berlin“ steht.**
