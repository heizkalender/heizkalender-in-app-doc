# Google API Key

Der Google API Key dient der sicheren Authentifizierung und Autorisierung der CCU gegenüber dem google Kalender.

Der API Key wird in der CCU als Systemvariable gespeichert: **HK1-GK-API-Key**

## Hinweis

Den Google API Key erstellen Sie wie folgt:

1) Bei Google (https://calendar.google.com) einen Kalender anlegen und z.B. „Belegungsplan“ benennen.

2) Freigabe: links: Im Feld *Meine Kalender* den Kalender auswählen

   ➔ drei Punkte ➔ auf Einstellungen und Freigabe klicken.

3) Person: links: *Für bestimmte Pers. oder Gruppen freigeben* wählen 

   ➔ Person hinzufügen ➔ mit dem Recht *Termine ändern*.

4) Rechte: links: *Zugriffsberechtigung für Termine* wählen 

   ➔ *öffentlich freigeben* wählen
   
5) **Freigabe-Link**: Darunter auf Feld *Link zum Freigeben abrufen* klicken 
   
   ➔  es erscheint ein Fenster mit dem Link ➔  kopieren
