# Gewünschte Raumtemperatur

Hier die gewünschte Wohlfühltemperatur hinterlegen.

Bitte beachten Sie, dass bei großem Besuch jede Person 70 Watt Heizleistung liefert.

Hinweis: Eine davon abweichende Wohlfühltemperatur kann beim Buchen im Kalender (nicht bei ChurchDesk) eingegeben werden, z.B. wenn ausnahmsweise ohne Heizung gebucht werden soll.
