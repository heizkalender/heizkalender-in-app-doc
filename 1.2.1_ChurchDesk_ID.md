# ChurchDesk ID

Die ChurchDesk ID Ihrer Gemeinde, so wie Sie in der Adresszeile (URL) zu ChurchDesk nach erfolgreichem Login steht.

Die ID wird in der CCU als Systemvariable gespeichert: **HK1-ICS-CD-Churchdesk-ID**

## Beispiel

Die ChurchTools URL *https://app.churchdesk.com/o/7025/dashboard* ergibt die ID **7025**.
