# DWD Station

Die geeignete Wetterstation des DWD finden Sie im Internet auf den Seiten des DWDs.

z.B.

https://www.dwd.de/DE/leistungen/klimadatendeutschland/stationsliste.html (Spalte Stationskennung)

oder

https://www.dwd.de/DE/leistungen/klimadatendeutschland/stationsuebersicht.html (Spalte Stationskennziffer)

Standardwert ist Frankfurt/M: 10637.

Der Wert wird in der CCU als Systemvariable gespeichert: **HK2-Location**
