# Google Kalender ID

Die Kalender ID ist eine eindeutige Kennung, die einen spezifischen Kalender innerhalb eines Google-Kontos identifiziert. 

Die ID wird in der CCU als Systemvariable gespeichert: **HK1-GK-Kalender-ID**

## Hinweis

Die Google Kalender ID erstellen Sie wie folgt:

1) Bei Google (https://calendar.google.com) einen Kalender anlegen und z.B. „Belegungsplan“ benennen.

2) Freigabe: links: Im Feld *Meine Kalender* den Kalender auswählen

   ➔ drei Punkte ➔ auf Einstellungen und Freigabe klicken.

3) Person: links: *Für bestimmte Pers. oder Gruppen freigeben* wählen 

   ➔ Person hinzufügen ➔ mit dem Recht *Termine ändern*.

4) Rechte: links: *Zugriffsberechtigung für Termine* wählen 

   ➔ *öffentlich freigeben* wählen

5) **Kalender ID**: links: *Kalender integrieren* wählen 

   *Kalender ID* suchen ➔ herauskopieren und abspeichern.

## Beispiel

5584c1afa9cb8169xxxxxxxxxxxxx28454de8a5cc9cca2@group.calendar.google.com ergibt die Kalender ID *5584c1afa9cb8169xxxxxxxxxxxxx28454de8a5cc9cca2*
