# iCal URL

iCal ist ein standardisiertes Dateiformat für Kalenderdaten. Ob Ihr Online-Kalender eine ICS / iCal Schnittstelle anbietet und wie Sie diese erstellen, erfahren Sie in der Dokumentation des Anbieters.

Die iCal URL wird in der CCU als Systemvariable gespeichert: **HK1-ICS-Url] **

## Beispiel: Microsoft Outlook und Exchange

Wenn Ihr Exchange-Server dies erlaubt, können Sie Ihren Belegungsplan-Kalender als Internetkalender mit einer iCal URL veröffentlichen und mit dem Heizkalender nutzen:

1) Im Outlook *Datei-Menü → Informationen → Kalender veröffentlichen* auswählen.

2) Den Belegungs-Kalender auswählen.

3) Outlook generiert eine iCal-URL. 

4) Den Link kopieren und hier einfügen.


## Beispiel: Microsoft 365 und Outlook im Web

1) Outlook.com öffnen und zur Kalender-Ansicht wechseln.

2) Klicken Sie oben rechts auf das Zahnrad ⚙️ → *Alle Outlook-Einstellungen anzeigen*.

3) → Kalender → Freigegebene Kalender.

4) den Belegungsplan-Kalender auswählen.

5) Unter Freigeben:

   - Einen Link zum Anzeigen erzeugen → das ist die iCal URL.
   
   - Den Link kopieren und hier einfügen.
   
   - Der Link beginnt meist mit webcal:// oder https://outlook.live.com/owa/calendar/....
